# Simple Expression Parser

Simple expression parser using Yacc, Flex, Lex and Bison

## Usage

To compile:

```bash
$ make
```

To compile and run:

```bash
$ make run
```

To compile and run tests:

```bash
$ make test
```

To clean up files:
```bash
$ make clean
```

## Tests

To test the output of the parser:

```bash
$ ./tests.sh
```
